Iterating Array Methods

All array iteration methods have a similar syntax. The original array, the method method call and the callback function as the method argument.

Syntax

        const array = ['🍎', '🍉', '🍒']
        array.method((item, idx, arr) => {
            // logic that will be used in each iteration
        });
  • item represents the individual item
  • idx represents the index number
  • arr is the array itself
Note: Most of the time we only pass 2 inside the parameters to get each element and index number

map() Method

The map(callback) method is used to transform an array. It calls a callback for each element of the original array and writes its result to a new array

    const fruits = ['🍉', '🍎', '🍌']
    const updatedFruits = fruits.map((fruit) => fruit + '🍌')
    updatedFruits // ['🍉🍌', '🍎🍌', '🍌🍌']
Note: The map method does not directly change the original array instead it creates a new updated array.

filter() Method

The filter(callback) method is used for one operation only – array filtering, that is, when more than one element from a collection should be selected using a specific criterion.

        const fruits = ['🍉', '🍎', '🍌']
        const updatedFruits = fruits.filter((fruit) => fruit !== '🍌')
        updatedFruits // ['🍉', '🍎']
Note: The filter method does not directly change the original array instead it creates a new updated array.

find() Method

The find method returns you that first item that matches the condition. If no item satisfy the condition then it returns undefined

        const fruits = ['🍉', '🍎', '🍌']
        const updatedFruits = fruits.find((fruit) => fruit === '🍌')
        updatedFruits // '🍌'

findIndex() Method

By the name of the method itself it returns you the index of the item you're looking for.

        const fruits = ['🍉', '🍎', '🍌']
        const index = fruits.find((fruit) => fruit === '🍌')
        index // 2

every() Method

Checks if all array elements pass the test provided by the callback. Returns true if all the items satisfy the condition or else false.

        const numbers = [10, 35, 42, 88, 5, 76]
        const result = numbers.every((number) => number > 0)
        result // true

some() Method

Checks if some of the array elements pass the test provided by the callback. Returns true atleast one satisfies the condition. If all the items doesn't satisfy the condition it returns false. 

        const numbers = [-10, -35, -42, -88, 5, -76]
        const result = numbers.some((number) => number > 0)
        result // true

sort() Method

The sort() method sorts the elements of an array, but unlike other iterating methods, it sorts the original array in place without having to create a copy.

        const numbers = [61, 19, 74, 35, 92, 56]
        numbers.sort()
        numbers // [19, 35, 56, 61, 74, 92]

However, since by default the values are converted to a string, the standard sorting of numbers works in an unusual way. Therefore, in the next exercise, let's look at how to set a custom sort order. 

        const scores = [27, 2, 41, 4, 7, 3, 75];
        scores.sort();
        scores; // [2, 27, 3, 4, 41, 7, 75]

        Sort by ascending formula a-b
        Sort by descending formula b-a

        const scores = [27, 2, 41, 4, 7, 3, 75];
        scores.sort((a, b) => a - b);
        scores   // [2, 3, 4, 7, 27, 41, 75]

        scores.sort((a, b) => b-a);
        scores // [75, 41, 27, 7, 4, 3, 2]

flatMap() Method

The flatMap(callback) method is similar to the map() method, but it is used in cases where the result is a multidimensional array or an array of objects that needs to be "flattened". 

        const students = [
            { name: "Mango", courses: ["mathematics", "physics"] },
            { name: "Poly", courses: ["informatics", "mathematics"] },
            { name: "Kiwi", courses: ["physics", "biology"] },
        ];

        Result of using map:
        students.map(student => student.courses);
        [['mathematics', 'physics'], ['informatics', 'mathematics'], ['physics', 'biology']]

        Result of using flatMap:
        students.flatMap(student => student.courses);
        ['mathematics', 'physics', 'informatics', 'mathematics', 'physics', 'biology']

reduce() Method

The reduce(callback, initialValue) method is used to process each array element, one by one, storing the intermediate result as an accumulator.

        const scores = [2, 7, 3, 14, 6]
        const result = scores.reduce((acc, score) => acc + score,0)
        result // 32
